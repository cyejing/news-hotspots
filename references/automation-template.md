# 定时任务模板

本文档提供每日/每周热点任务的提示词模板，执行细节和占位符说明见 `execution-guide.md`。

---

## 每日热点

```text
阅读 <SKILL_DIR>/references/execution-guide.md，使用以下参数执行每日热点任务：

- MODE = daily
- RSS_HOURS = 48
- LANGUAGE = Chinese
```

---

## 每周热点

```text
阅读 <SKILL_DIR>/references/execution-guide.md，使用以下参数执行每周热点任务：

- MODE = weekly
- RSS_HOURS = 168
- LANGUAGE = Chinese
```
